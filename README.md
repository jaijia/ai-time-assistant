# AI 时间管理助理

> 基于人工智能的智能时间管理工具，通过自然语言交互帮助用户高效规划日程安排

**开发团队：黄枚楣小组**

## 📋 项目简介

AI 时间管理助理是一个现代化的全栈 Web 应用程序，采用前后端分离架构，结合 DeepSeek AI 模型，为用户提供智能化的时间管理解决方案。系统能够理解自然语言输入，自动解析任务信息，检测时间冲突，并提供个性化的时间管理建议。

## 🚀 快速开始

### 环境要求
- Python 3.8 或更高版本
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- DeepSeek API 密钥

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd ai_time_assistant
   ```

2. **创建虚拟环境**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS/Linux
   source venv/bin/activate
   ```

3. **安装依赖**
   ```bash
   pip install -r backend/requirements.txt
   ```

4. **配置环境变量**
   ```bash
   # 复制示例文件
   cp backend/.env.example backend/.env
   # 编辑 .env 文件，添加你的 DeepSeek API 密钥
   DEEPSEEK_API_KEY="your_api_key_here"
   ```

5. **启动后端服务**
   ```bash
   python -m backend.app
   ```

6. **打开前端页面**
   - 在浏览器中打开 `frontend/index.html`
   - 或使用本地服务器：`python -m http.server 8000`

## 🔧 API 接口

### 任务管理
- `GET /api/tasks` - 获取所有任务
- `POST /api/tasks` - 创建新任务（通过 AI 解析）
- `PATCH /api/tasks/<id>` - 更新任务状态或信息
- `DELETE /api/tasks` - 清空所有任务
- `DELETE /api/tasks/<id>` - 删除指定任务

### 系统状态
- `GET /` - 根路径，返回服务状态
- `GET /health` - 健康检查

## 💡 使用指南

### 基本使用
1. 在文本框中输入你的任务描述，例如：
   - "明天上午9点到11点写项目报告，优先级高"
   - "下午2:30-3:30开会讨论新功能"
   - "晚上7点-8点健身房锻炼"

2. 点击"提交给 AI 规划"按钮

3. AI 会自动解析任务并添加到日程表中

4. 查看 AI 提供的优化建议

### 快速操作
- 使用快速操作按钮快速填充常用模板
- 点击任务行的复选框切换完成状态
- 点击任务行的删除按钮删除单个任务
- 使用"清空所有"按钮批量删除任务
- 点击"导出日程"下载 CSV 文件

### 任务管理
- 查看总任务数和已完成任务数统计
- 点击复选框标记任务为已完成
- 过期任务自动显示为横杠和浅色样式
- 实时更新任务状态和统计数据
